2. The Agent "Soul" (Durable Object)
Using the Agents SDK (v0.8.0), each user gets a dedicated Durable Object. This is where the OpenClaw memory (MEMORY.md) lives, stored in the DO’s internal SQLite database.

TypeScript
import { Agent } from "@cloudflare/agents";

export class OpenClawAgent extends Agent {
  async onStart() {
    // Initialize OpenClaw memory in SQLite if it doesn't exist
    await this.sql`CREATE TABLE IF NOT EXISTS memory (key TEXT PRIMARY KEY, content TEXT)`;
  }

  async spawnSandbox(task: string) {
    // Retrieve the AGENTS.md and SKILLS.md from this agent's state
    const skills = await this.sql`SELECT content FROM memory WHERE key = 'skills'`;

    // 1. Create the Sandbox on-the-fly (The "Body")
    const sandbox = await this.env.LOADER.load({
      mainModule: "worker.js",
      modules: {
        "worker.js": `
          export default {
            async fetch(req) {
              const { task, skills } = await req.json();
              // THE OPENCLAW RUNTIME GOES HERE
              // Execute the task using the provided skills
              return new Response(JSON.stringify({ output: "Task Complete" }));
            }
          }
        `
      },
      env: { AI: this.env.AI_BINDING }
    });

    // 2. Execute the task
    return await sandbox.getEntrypoint().fetch(new Request("http://sandbox/", {
      method: "POST",
      body: JSON.stringify({ task, skills })
    }));
  }
}
3. The Parallel Agent Dashboard (Frontend)
Don't build a custom WebSocket handler. Use the new @cloudflare/sandbox/xterm library to connect the browser directly to the agent's PTY (Pseudo-Terminal) stream.

Frontend Component (React + Vite)
TypeScript
import { Terminal } from "@xterm/xterm";
import { SandboxAddon } from "@cloudflare/sandbox/xterm";

function AgentTerminal({ agentId }) {
  const terminalRef = useRef(null);

  useEffect(() => {
    const term = new Terminal({ theme: { background: '#1a1b26' } });
    const addon = new SandboxAddon({
      // This calls your Orchestrator Worker to get a signed WebSocket URL
      getWebSocketUrl: async () => {
        const res = await fetch(`/api/agent/${agentId}/connect`, { method: 'POST' });
        return (await res.json()).url;
      }
    });

    term.loadAddon(addon);
    term.open(terminalRef.current);
    addon.connect({ sandboxId: agentId });
    
    return () => term.dispose();
  }, [agentId]);

  return <div ref={terminalRef} className="h-96 w-full" />;
}

5. Why this is superior to the "AI's version"
True Isolation: The env.LOADER.load() creates a fresh V8 isolate. If an agent tries to run rm -rf /, it only destroys its own ephemeral sandbox, not your server.

Scale to Millions: You don't pay for idle servers. If 10,000 users aren't typing in their terminals, you pay $0.

Zero Latency: The "boot time" for a new parallel agent is ~5ms. To the user, it feels like the agent is "always on."
